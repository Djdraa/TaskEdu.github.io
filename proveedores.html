💼 Análisis de Proveedores

Evaluación técnica y económica para la implementación del proyecto

Comparativo de Proveedores

Proveedor Infraestructura Software Costo Total (COP)
Modelo Tradicional Empresarial Servidor físico dedicado Windows Server, SQL Server $7.000.000 – $9.000.000
Modelo Académico Open Source SELECCIONADO Equipo personal / Hosting en la nube Node.js, Express, MongoDB $0 – $360.000

Conclusión del Análisis

Tras el análisis comparativo entre un modelo tradicional con infraestructura física y el modelo académico basado en tecnologías de código abierto, se concluye que el Modelo Académico Open Source es la opción más adecuada.

Este enfoque elimina la necesidad de adquirir servidores físicos y pagar licencias comerciales, reduciendo significativamente los costos iniciales. Además, permite una arquitectura moderna basada en Node.js, Express y MongoDB, garantizando flexibilidad, escalabilidad y facilidad de mantenimiento.

Stack Tecnológico Seleccionado

  • Lenguaje: JavaScript
  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Node.js, Express.js
  • Base de datos: MongoDB / PostgreSQL (Neon Console)
  • Hosting / Deploy: Render (servidor en la nube)
  • Monitoreo: UptimeRobot
  • Autenticación: JSON Web Tokens (JWT)
  • Control de versiones: Git y GitHub
  • Editor: Visual Studio Code
  • Metodología: Scrum (sprints semanales, entregas iterativas)
